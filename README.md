How to run the test build of the egg counter system locally
 

1.) Run Server.bat located in the main folder, wait for the terminal to finish loading 

2.) Navigate to the EggCounter Folder, and run Start_Expo.bat to run the expo front-end server

3.) In the Expo terminal, press W to run the front-end in the browser. 

Note; 

If both .bat files prompts you with a "File not found" or "directory not found" error, clone the files and manually change the Server.bat and Start_Expo.bat files to the correct location of the files needed. Use notepad or any similar editors.
